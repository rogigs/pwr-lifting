## Run Locally

Clone the project

```bash
git@github.com:rogigs/pwr-lifting.git
```

Go to the project directory

```bash
  cd pwr-lifting
```

Install dependencies

```bash
  npm install
```

Start project

```bash
  npm run start
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
